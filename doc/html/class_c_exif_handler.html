Detailed Description

Klassen ansvarar för att räkna upp antalet bilder som används i montaget och räknar ut den totala slutartiden (i sekunder). För att minimera io så öppnas bara sista filen i serien för läsning av exif, sedan multipliceras slutartiden med antalet bilder och den nya beräknade slutartiden.
